





Rest Api  blog crud operations  
====================================================



Preparation 
------------
```
npm  install
```


Starting application 
--------------------
```
 node index.js
```


API endpoints
--------------------
```
https://localhost:4000/api
```

`/api/blog` Get - Get blogs.

`/api/blog` Post - Create blog.

`/api/blog/:id` Put - Put blog.

`/api/blog/:id` Delete - Delete single Blog.
